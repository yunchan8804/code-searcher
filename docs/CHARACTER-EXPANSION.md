## 품질 기준

### 각 문자 항목 필수 정보
```json
{
  "char": "★",           // 필수: 실제 문자
  "codepoint": "U+2605", // 필수: 유니코드 코드포인트
  "name": "BLACK STAR",  // 필수: 공식 유니코드 이름
  "tags_ko": ["별"],     // 필수: 한글 검색 태그 (1개 이상)
  "tags_en": ["star"],   // 필수: 영어 검색 태그 (1개 이상)
  "category": "star",    // 필수: 카테고리 ID
  "frequency": 95        // 필수: 사용 빈도 (1-100)
}
```

### 태그 작성 가이드
- 한글 태그: 일상적으로 사용하는 단어 (예: "별", "하트", "화살표")
- 영어 태그: 공식 이름 + 일반적 명칭 (예: "star", "heart", "arrow")
- 동의어/관련어 포함 (예: "체크" + "확인" + "완료")
